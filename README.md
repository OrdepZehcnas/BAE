# BAE
Para compilarlo deben tener GHC y hacer lo siguiente:

  Donde descompriman el zip, ejecuten:
            
                                              ghc -o BAE Main.hs
  
  Una vez compilado el programa, pueden ejecutarlo con los programas ejemplos, es decir:
                                              
                                              ./BAE goodExample.bae
                                              ./BAE badExample.bae
